Defines very simple corner appearance options for entire Surface.

BEVEL → const SurfaceCorners

Only two diagonally-oppposite corners will be beveled by radius, while the other two remain square. Mirror the shape with bool parameter flipBevels.

ROUND → const SurfaceCorners

All four corners will be rounded. This is default, and the default value is Surface._DEFAULT_RADIUS.

const SurfaceCorners(1)
